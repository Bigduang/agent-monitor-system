# Agent 监控系统

## 技术选型

### 前端
- Vue 3 + TypeScript
- Vite (构建工具)
- pnpm (包管理器)
- TailwindCSS (UI 框架)

### 后端
- PHP 8 + Laravel 11

### 部署
- LNMP 架构 (Linux + Nginx + MySQL + PHP)
- 无需 Docker

### 任务管理
- GitHub Issue (纯 GitHub 管理，无需本地数据库)

### 认证
- 内网项目，无需登录注册，直接访问 Dashboard

---

## 目录

- `frontend/` - 前端项目
- `backend/` - 后端项目
